# Change Log

All notable changes to the "Cyberpunk 2077" extension will be documented in this file.

## [1.0.0] - 2026-02-13

### Added

- Dark theme with Cyberpunk 2077 color palette
- Full UI theming: editor, sidebar, activity bar, tabs, status bar, terminal, buttons, inputs, scrollbar, breadcrumbs, panels, peek view, minimap
- Syntax highlighting using 9 colors from the Cyberpunk 2077 palette
- Git decoration colors
- JSON key coloring by nesting level (up to 8 levels)
- Markdown styling
- Terminal ANSI color mapping
- Neon green comment color (#00ff9f)
